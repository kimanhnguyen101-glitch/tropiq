01Tổng quan

Website marketing cho Tropi Q Head Spa — spa cao cấp tại Hà Nội, 85% khách du lịch quốc tế.

  • Ngôn ngữ: Tiếng Anh (v1). Định hướng v2 thêm Việt/Trung/Nhật.
  • Mục tiêu: SEO mạnh để giảm phụ thuộc quảng cáo trả phí; cảm giác tropical, cao cấp; đặt lịch qua WhatsApp.
  • Tech: Next.js 16 + React 19 + Tailwind CSS v4. Không backend, không database — nội dung tĩnh trong code.
  • Triển khai: Vercel (đã live, công khai).

02Tiến độ dự án

Các mốc đã hoàn thành và việc còn lại.

✅ Đã hoàn thành

  • Dựng website v1 — 9 trang: Trang chủ, About, Services, Head Spa, Experience, Reviews, Franchise, Journal (blog khung), Contact.
  • SEO nền tảng — metadata từng trang, 1 H1/trang, JSON-LD (HealthAndBeautyBusiness + rating 5.0), sitemap, robots, og:image.
  • Áp dụng feedback V1 — màu/font mới, cover ảnh mỗi trang, cover trang chủ chỉ ảnh + H1 ở dưới, bỏ chữ "ritual", "5.0★ guest rated", "Find your perfect match", giờ mở cửa 10:30–20:30, khu video placeholder, card acceptance.
  • Gắn logo thật — header (olive), footer (cream), favicon (chữ Q).
  • Deploy công khai trên Vercel (đã tắt Deployment Protection).

⏳ Đang chờ tư liệu

  • Cần xử lý Bộ ảnh mới — 87 ảnh đang ở _source/new-photos-raw-heic/ (định dạng HEIC, cần convert sang JPG rồi thay vào site).
  • Cần file Video — khu video khách trải nghiệm (trang chủ) + video minh hoạ từng nhóm dịch vụ (hiện là placeholder).

📋 Việc tiếp theo

  • Ưu tiên Gắn domain tropiqspa.vn (Vercel ▸ Settings ▸ Domains). Sau đó canonical/OG/JSON-LD (đang trỏ tropiqspa.vn) khớp 100% & ảnh share Zalo/Facebook hoạt động.
  • Submit sitemap lên Google Search Console + đồng bộ Google Business Profile.
  • v2: đa ngôn ngữ, nội dung blog thật, embed Google reviews thật, mục 10 Fingers Nail.

03Chạy dự án (local)

Yêu cầu: Node.js ≥ 20.

npm install        # cài thư viện (lần đầu)
npm run dev        # chạy thử → http://localhost:3000
npm run build      # build production (chạy để kiểm tra trước khi deploy)
💡 Đồng nghiệp dùng Claude Code: repo đã có sẵn file CLAUDE.md ở thư mục gốc — Claude sẽ tự đọc để hiểu kiến trúc & quy ước. Cứ mở Claude trong thư mục này và yêu cầu việc cần làm.

04Deploy lên Vercel

Dự án đã link sẵn với project Vercel tên tropiq (thư mục .vercel/). Deploy bằng đúng tài khoản Vercel đang giữ project này.

npx vercel login          # đăng nhập tài khoản Vercel giữ project tropiq
npx vercel --prod --yes   # deploy production → tự alias về tropiq-seven.vercel.app
⚠️ Đừng tạo project mới. Nếu lỡ tạo mới, Vercel sẽ bật lại "Deployment Protection" khiến web đòi đăng nhập — vào Settings ▸ Deployment Protection để tắt.

Thêm ảnh mới (từ HEIC)

Ảnh gốc ở _source/new-photos-raw-heic/. Convert sang JPG rồi bỏ vào public/images/store/:

# macOS có sẵn lệnh sips
sips -s format jpeg -Z 1800 "anh.HEIC" --out "public/images/store/store-XX.jpg"

Gắn video

Dùng component <VideoEmbed>: truyền src="/videos/ten.mp4" (bỏ file vào public/videos/) hoặc embedUrl="..." (YouTube/Vimeo). Video từng nhóm dịch vụ: set video/embedUrl cho group trong src/lib/menu.js.

07Lưu ý quan trọng

  • Tailwind v4 dùng @theme trong CSS — không có tailwind.config.js.
  • Dùng JavaScript (không TypeScript). Ảnh luôn dùng next/image.
  • Token màu gold thực chất là màu terracotta #B56B2C (giữ tên cũ để khỏi sửa nhiều nơi).
  • Mỗi trang giữ đúng 1 thẻ H1 và metadata riêng — đừng phá vỡ khi sửa (ảnh hưởng SEO).
  • Số "88 reviews" cố tình ẩn trên giao diện nhưng giữ trong JSON-LD để Google hiện sao vàng — đừng xoá khỏi schema.
  • Thư mục _source/ chỉ là tư liệu gốc, không deploy lên web.
