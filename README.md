# Tropi Q — Head Spa Website

Website marketing (tiếng Anh) cho Tropi Q Head Spa, Hà Nội. Next.js 16 + React 19 + Tailwind v4.
Live: https://tropiq-seven.vercel.app

## Bắt đầu nhanh
```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # build production
```

## Tài liệu bàn giao
- **`HUONG-DAN-BAN-GIAO.html`** — hướng dẫn sử dụng đầy đủ + tiến độ dự án (mở bằng trình duyệt).
- **`CLAUDE.md`** — hướng dẫn kiến trúc & quy ước cho Claude Code (tự đọc khi mở project).

## Deploy
Project đã link sẵn Vercel (`tropiq`). Đăng nhập đúng tài khoản rồi:
```bash
npx vercel --prod --yes
```

## Tư liệu gốc
Thư mục `_source/` chứa logo, menu PDF, ảnh mới (HEIC chưa xử lý), file feedback — KHÔNG deploy lên web.
